Comment utiliser les paramètres:

Les URLs suivantes démontrent comment passer des paramètres:

  • /planete?id=3 - Affiche la Terre
  • /satellite?id=1 - Affiche la Lune

Liste des planètes:

    {% for planet in planets %}
  • {{ planet.name }} (ID: {{ planet.id }}) - Exemple: /planete?id={{ planet.id }}
    • {% endfor %}

Explications pour les élèves:

Dans l'URL /planete?id=3 :

  • id est le nom du paramètre
  • 3 est la valeur du paramètre
  • Dans Flask, on récupère cette valeur avec request.args.get('id')
